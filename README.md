Smart Remittance Integration Platform



A full-stack integration system for foreign workers to send money with real-time currency conversion, built using modern enterprise integration architecture.



Features

\- Currency conversion

\- Money transfer simulation

\- Integration APIs

\- Observability dashboard



Tech Stack

\- Node.js

\- Express

\- WSO2 / Ballerina (planned)

\- React (planned)
